# devcontainer

A personal development container image published to the GitHub Container Registry.

## Image

The image is available on GHCR:

```
ghcr.io/marcopist/devcontainer:latest
```

[View on GitHub Container Registry](https://ghcr.io/marcopist/devcontainer)

## Usage

```sh
docker pull ghcr.io/marcopist/devcontainer:latest
```
